# Ping Pong in BCFD
Welcome to the source code! Feel free to use these scripts in any project you want.

__Steps:__
1. Create these commands:
- request (Phrase ON, Reaction: ✅)
- request (Phrase ON, Reaction: ❌)
- ping (Phrase ON)
- finish (Phrase ON)
- ✅ (Event: Reaction Add)
- ❌ (Event: Reaction Add)
- help

2. Insert scripts - don't forget to follow the instructions on each one!
